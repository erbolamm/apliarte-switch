¿Qué es?

ApliArteSwitch es un servicio que corre en segundo plano y te permite mover el ratón al borde de la pantalla para “saltar” al siguiente ordenador de tu red local.

  • Sin hardware extra (sin dongles, sin cables).
  • En red local (WiFi/LAN).
  • UI opcional para configurar (Flutter Desktop).

Estado (importante)

Este proyecto está en desarrollo (WIP). La fase actual se centra en macOS. Windows y Linux están planificados.

Ya planteado

  • Switch automático por borde.
  • Hotkey manual (Ctrl+Alt+Tab por defecto).
  • Descubrimiento en LAN (mDNS).

Pendiente

  • Portapapeles compartido.
  • Cifrado DTLS (en progreso / por cerrar).
  • Empaquetado fácil por plataforma.

Instalación

Atajo: instala el binario o compila desde código.

macOS / Linux

curl -fsSL https://raw.githubusercontent.com/erbolamm/apliarte-switch/main/scripts/install.sh | bash

Windows (PowerShell)

irm https://raw.githubusercontent.com/erbolamm/apliarte-switch/main/scripts/install.ps1 | iex

Con Cargo (todas)

cargo install apliarte-switch

Arquitectura

El proyecto está dividido en dos piezas:

  • Core (Rust): motor/servicio (input + red + IPC).
  • UI (Flutter Desktop): panel de control para configurar.
